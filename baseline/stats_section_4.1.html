Section 4.1: Overall Performance Summary — Full Details

Overview

Section 4.1 provides the definitive CytoSig vs SecAct summary across 6 datasets at each dataset’s primary independent level. This supplement includes all aggregation levels (not just the primary independent level shown in the main report).

Independence-Corrected Methodology

All correlations use a median-of-medians approach to ensure statistical independence:

  • GTEx (by_tissue, 29 tissues): Each target’s representative ρ = median of its per-tissue Spearman ρ values across 29 tissues.
  • TCGA (primary_by_cancer, 33 cancer types): Same median-of-medians approach across 33 cancer types.
  • CIMA, Inflammation Main, scAtlas Normal, scAtlas Cancer: Each target has one ρ at the donor/tumor level (no tissue/cancer stratification needed), so no median-of-medians aggregation is required.

% Sig: Benjamini–Hochberg FDR correction (q < 0.05) applied within each dataset × method group.
